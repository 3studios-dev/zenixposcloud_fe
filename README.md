## Style Guide

### Theme and Color Scheme

- **Primary Background**: Black (`#000000`)
- **Secondary Background**: Dark Gray (`#111827`)
- **Card Background**: Darker Gray (`#1F2937`)
- **Accent Color**: Green (`#22C55E`)
- **Text Colors**:
  - Primary: White (`#FFFFFF`)
  - Secondary: Gray (`#9CA3AF`)
- **Border Color**: Dark Gray (`#374151`)

### Typography

- **Headers**: Bold, White
- **Body Text**: Regular, Gray
- **Prices**: Bold, Green
- **Buttons**: Bold, White on Green

### Layout

- **Grid System**: Flex-based responsive grid
- **Spacing**: Consistent 16px (1rem) base unit
- **Borders**: Rounded corners (8px)
- **Icons**: Lucide React Native icons

## Technologies Used

- React Native
- NativeWind (Tailwind CSS for React Native)
- Expo Router
- Lucide React Native Icons

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/ristouch-pos.git
```

2. Install dependencies:
```bash
cd ristouch-pos
npm install
```

3. Start the development server:
```bash
npm start
```

## Usage

1. Launch the app on your device or emulator
2. Log in with your credentials
3. Select the appropriate menu category
4. Add items to the current order
5. Apply any discounts or modifications
6. Process payment and complete the order
